Comment Forger une MCE Artificielle

1
La Cible
Le Northbridge
Contrôleur qui gère la mémoire et les périphériques PCIe
2
Méthode
Fuzzing des registres de configuration du Northbridge
Test automatisé de différentes valeurs
3
Le Résultat
Configuration "empoisonnée"
Transforme une erreur bénigne en MCE fatale
4
Le Contrôle
L'attaquant peut déclencher un crash matériel à volonté
Via une simple ligne de code
